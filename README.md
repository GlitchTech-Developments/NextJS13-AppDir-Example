## Install

```sh
pnpm install
```

## Usage

> **AppName**: package name of project of choice in apps directory
> The NextJS project is under the "web" name. So filter on web when targeting the NextJS project.

```sh
pnpm run dev --filter [AppName]
```

## Run tests

```sh
pnpm run test
```
